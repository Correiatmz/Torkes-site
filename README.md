## Disciplina

Front-End

## Projeto

Site institucional da marca fictícia de veículos Torkes Motors.

---

# Tema Escolhido

O projeto desenvolvido consiste em um site automotivo fictício da marca Torkes Motors.

A proposta foi criar uma experiência visual moderna inspirada em sites reais de montadoras premium, apresentando dois modelos de veículos:

* Torkes K1S
* Torkes K7

O site foi desenvolvido utilizando HTML5, CSS3 e Bootstrap 5, com foco em responsividade, organização visual e experiência do usuário.

## Explicação das páginas

### index.html

Página principal do site.

Contém:

* Navbar fixa
* Carousel de imagens
* Apresentação da marca
* Cards dos modelos
* Sessão sobre a empresa

### k1.html

Página dedicada ao modelo Torkes K1S.

Contém:

* Hero section
* Galeria do veículo
* Informações técnicas
* Modal de orçamento
* Sessão de tecnologia

### k7.html

Página dedicada ao modelo premium Torkes K7.

Contém:

* Destaques do veículo
* Interior premium
* Performance
* Modal de reserva
* Galeria de imagens

### contatos.html

Página de contato e concessionárias.

Contém:

* Concessionárias fictícias da região Sul do Brasil
* Formulário de contato
* FAQ utilizando Accordion Bootstrap

# Componentes Bootstrap Utilizados

Durante o desenvolvimento foram utilizados diversos componentes do Bootstrap 5 para melhorar a responsividade e organização visual do site.

## Componentes utilizados

### Navbar

Utilizada para criar o menu de navegação responsivo.

### Dropdown

Utilizado dentro da navbar para separar os modelos da marca.

### Carousel

Utilizado na página principal para exibição de imagens dos veículos.

### Cards

Utilizados para apresentar os modelos e concessionárias.

### Modal

Utilizado para formulários de orçamento e agendamento.

### Accordion

Utilizado na página de contatos para perguntas frequentes.

### Grid System

Utilizado para organização responsiva das páginas.

# Decisões de Layout e Design

O layout foi desenvolvido inspirado em sites modernos de montadoras premium.

## Principais decisões visuais

* Uso predominante de cores escuras
* Navbar transparente fixa
* Imagens em destaque ocupando grande parte da tela
* Visual minimalista e tecnológico
* Uso de espaçamento amplo para melhor leitura
* Cards com efeitos de hover
* Layout responsivo para celulares e desktops

O objetivo foi transmitir modernidade, luxo e tecnologia.
